Display and manage your website server.

== Description ==

Display useful information of your web site server. Keep under control your server status.

= Features =

* Display your web server information
* Display process
* Display the CPU stats
* Display the RAM stats

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the entire content of plugin archive to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress (deactivate and reactivate if you're upgrading).
3. Done. Enjoy.
